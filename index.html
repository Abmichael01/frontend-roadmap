FRONTEND

ROADMAP

1

Foundations

Internet Basics:

  • Understand how the internet works.
  • Learn about HTTP, browsers, DNS, and domain names.
  • Explore hosting concepts.

HTML (Hypertext Markup Language):

  • Learn the basics of HTML.
  • Write semantic HTML for better accessibility.
  • Understand forms and validation.

CSS (Cascading Style Sheets):

  • Create layouts using CSS.
  • Dive into responsive design.
  • Explore CSS preprocessors (e.g., Sass) and architecture (e.g., BEM).

2

JavaScript and Frameworks

JavaScript:

  • Master the fundamentals of JavaScript.
  • Learn about the Document Object Model (DOM) manipulation.
  • Understand Fetch API and Ajax (XHR).

Version Control Systems (VCS):

  • Get comfortable with Git and GitHub.
  • Understand version control concepts.

Frameworks:

  • Choose a framework (React, Angular, Vue.js, Svelte, or Solid JS).
  • Explore CSS-in-JS libraries (e.g., Styled Components, Emotion).
  • Use linters and formatters (e.g., Prettier, ESLint).

3

Advanced Topics

TypeScript:

  • Dive into TypeScript for type-safe JavaScript.
  • Learn about type checkers and static typing.

Server-Side Rendering (SSR):

  • Explore SSR frameworks (e.g., Next.js, Nuxt.js).
  • Understand server-side rendering concepts.

Progressive Web Apps (PWAs):

  • Learn about service workers, storage, and notifications.
  • Understand PRPL pattern and performance metrics.
